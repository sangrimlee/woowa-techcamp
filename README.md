## 🖥 Start

### 1. Clone and install package

```bash
git clone https://github.com/woowa-techcamp-2022/web-moneybook-2.git

npm install
```

### 2. Set environment variables

```bash
# ./server/.env

PORT=
DB_HOST=
DB_NAME=
DB_USER=
DB_PASSWORD=
```

### 3. Build and run app

```bash
# production
npm run start:prod

# development
npm run dev
```
